# utils-project

## 一个使用electron构建的跨平台工具箱,页面使用vue.js搭建
已完成功能

- 时间戳转换
- 身份证校验 解析 随机生成
- 正则表达式测试 

> An electron-vue project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
